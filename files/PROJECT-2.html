CoCo: A Collaborative Coding Platform

  • Complitation Date: 2024-05-28
  • Tech Stack: TypeScript
  • GitHub Reposotpry: click-here

This project explores the collaborativelearning environment at Aalborg University. It focuses on understandingand improving upon the current dynamics of doing coding exercises instudy groups. Research has shownthat currently at Aalborg University,there is no tool for facilitating collaborative coding in study groups. To tryand tackle this issue, CoCo has beendeveloped to enhance the collaborative coding experience among studygroups at AAU, while having the capabilities to congregate many differentcourses, lectures, and exercises on asingle web platform.

CoCo successfully provides real-time collaborative coding and instant feedback, smoothening the learning process for inexperienced coders. Using technologies such as Yjs and CodeMirror, CoCo ensures conflict free collaboration between users. Furthermore with an integration of Judge0 CoCo is able to execute and display the result of code written by users.
